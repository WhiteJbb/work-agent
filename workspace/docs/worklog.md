## 2025-06-19 — 원격 vLLM 연결

- 로컬에 GPU가 부족해 사내 서버의 vLLM 엔드포인트를 원격으로 붙임.
- OpenAI-compatible API라서 base_url만 바꿔 연결 시도.
- PowerShell에서 curl 테스트 시 `curl`이 `Invoke-WebRequest` alias라 헤더 옵션이 안 먹어 한참 헤맴. `curl.exe`로 직접 호출해 해결.
- 방화벽/포트 확인 후 연결됨.
