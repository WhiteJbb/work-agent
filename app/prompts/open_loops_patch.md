## 지시사항

위 기록을 분석해 `40_AgentMemory/05_OpenLoops.md`에 반영할 패치 후보를 만드세요.

- **추가할 항목**: 기록에서 미해결로 보이는 것 (아직 안 한 것, 막힌 것, 확인 필요한 것)
- **완료 처리할 항목**: 현재 Open Loops에 있지만 기록에서 완료된 것
- **보류할 항목**: 아직 시작하지 않았거나 미래로 미룬 것

### 규칙
- 기록에 근거가 있는 항목만 추가한다.
- 불확실하면 항목 끝에 "(확인 필요)"를 표시한다.
- 완료 처리는 확실히 끝난 경우만 한다.

### 반환 형식

아래 JSON 형식으로만 응답하세요:

```json
{
  "add": [
    {"item": "추가할 항목 설명", "source_ref": "근거 파일 경로", "priority": "high"}
  ],
  "complete": [
    {"item": "완료된 항목 설명 (기존 OpenLoops 내용과 유사하게)", "source_ref": "근거 파일 경로"}
  ],
  "defer": [
    {"item": "보류 항목", "reason": "보류 이유"}
  ],
  "rationale": "전체 분석 요약 (한두 문장)"
}
```

항목이 없는 섹션은 빈 배열 `[]`로 반환합니다.
